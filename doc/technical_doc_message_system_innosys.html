A description of the Web Service provided by Innosys Pte Ltd

A brief introduction to the Message System

  The Web Service provided by Innosys(the WS) is for the Transport Module users(the
Users)to interact with their drivers(the Drivers) as well as Growth Venture(the GV, which
provides PSA data).
  i.e. The User can send instructions to Drivers, receive port information from GV, and so on.

How to use this Web Service

  The WS works at background, the Users may make use of it without be aware of its
existence.
  The User just presses buttons on the TM as usual, and the TM makes use of the WS,
to send and receive messages to and from their Drivers.
  This is just a simple introduction, we'll a detailed user manual in future.
